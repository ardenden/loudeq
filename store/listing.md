**App name:** loudeq

**Short description:**
One-click toggle for Windows Loudness Equalization — from the system tray, a pinned taskbar button, or the command line.

**Description:**

Windows' built-in Loudness Equalization evens out volume differences: quiet sounds get louder, loud peaks get tamed. Great for movies at night, quiet dialogue, or weak USB speakers — but toggling it means digging through the Sound control panel every single time.

loudeq makes it one click:

- Tray icon always shows the current state: green dot = ON, gray ring = OFF
- Click the icon (or a pinned taskbar shortcut) to toggle
- Changes apply live to playing audio — no restarts, no admin prompts
- Optional "start with Windows"
- Command line included: loudeq on / off / toggle / status / list / meter
- Works with any playback device that supports Windows audio enhancements, including USB devices that vendor audio consoles ignore

Free and open source: https://github.com/ardenden/loudeq
